Quadruped Simulation (Work in Progress)

The goal of this masters' thesis is to simulate the B.R.U.C.E quadruped and use reinforecment learning to train an the robot to move in a walking motion. This learned walking motion will then be compared to a classical control theory example of a walking motion using a gait scheduler. Isaac Gym will be used as the simulation software in order to take advantage of the GPUs parallel computing capabilities and speed up training with out sacrificing accuracy. Various reinforcement learning algorithms will be used to train B.R.U.C.E. such as PPO and DDPG. Before however, reinforcement learning was applied to the simpler classical cart-pole problem in control theory to validate and create a benchmark for learning.
